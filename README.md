# mind-map
## 项目简介
基于vue2的脑图编辑器，同时接入了AI对话，可以通过AI编辑脑图节点的内容。
- [x] 脑图编辑
- [x] AI对话
- [x] AI编辑脑图节点内容
- [x] 脑图导出
- [ ] AI编辑脑图上下级关系

## Project setup
```
yarn install
```

### Compiles and minifies for production
